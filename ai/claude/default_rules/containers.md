## Containers

- Use Podman by default. If you think Docker makes more sense, let me know, then
  ask me to confirm.
- Every application you build should be executable in a container.

## Kubernetes

- ALWAYS ensure that the application you build works in Kubernetes.
- ALWAYS Use `kind` to test your application in Kubernetes locally.
- ALWAYS ASK ME before making any changes to a remote Kubernetes cluster. (You
  don't need to ask me to make changes to local Kubernetes clusters.)
